# 契約法学習ゲーム

春野瑛士弁護士と一緒に契約法を学ぶ対話型ゲームです。

## ゲームの概要

このゲームでは、弁護士の春野瑛士さんがあなたの契約法の理解をテストします。各フェーズで質問に答え、キーワードを含む回答をすることでハートゲージが上がります。

## 遊び方

### Web版（推奨）
1. `index.html` をブラウザで開く
2. 春野瑛士さんの質問に日本語で回答してください
3. 各フェーズのキーワードを含む回答をすることでハートゲージが上がります
4. 全4フェーズをクリアして最終的なハートゲージで結果が決まります

### コンソール版
```bash
npm start
```

## フェーズ

- **Phase 1**: 価値観に基づく判断
- **Phase 2**: 法原理の適用  
- **Phase 3**: ドラフト修正案ワーク
- **Phase 4**: 多様な判断基準の認識

## キーワード例

各フェーズで以下のようなキーワードを含む回答をすると成功します：

- Phase 1: 「不平等」「情報格差」「読む」「同意」
- Phase 2: 「売買契約」「対等」「承諾」「条文」
- Phase 3: 「質問」「明確」「時間」「格差」
- Phase 4: 「知識不足」「難解」「時間制約」「情報差」

## 結果

- 80点以上: 完璧！未来の法律を変えられる
- 50-79点: よく頑張った！あと少しで理想の改変案が完成
- 50点未満: 大丈夫、誰だって最初は戸惑うもの
